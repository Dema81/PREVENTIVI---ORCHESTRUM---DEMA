Sub GeneraLinkHTML() Dim ws As Worksheet Dim baseURL As String Dim quoteID As String Dim subject As String Dim languages As String Dim amount As String Dim senderEmail As String Dim cc As String Dim bcc As String Dim fullURL As String ' Imposta il foglio attivo Set ws = ThisWorkbook.ActiveSheet ' URL base del file HTML su GitHub baseURL = "https://dema81.github.io/PREVENTIVI---ORCHESTRUM---DEMA" ' Recupera i dati dalle celle di Excel (A1 a B5) quoteID = ws.Range("A1").Value subject = ws.Range("A2").Value languages = ws.Range("A3").Value amount = ws.Range("A4").Value senderEmail = ws.Range("A5").Value cc = ws.Range("B1").Value bcc = ws.Range("B2").Value ' Costruisci l'URL completo con i parametri fullURL = baseURL & "?quoteID=" & quoteID & "&subject=" & subject & "&languages=" & languages & "&amount=" & amount & "&sender=" & senderEmail & "&cc=" & cc & "&bcc=" & bcc ' Mostra l'URL generato MsgBox "Link generato: " & fullURL ' Apri il link nel browser predefinito Shell "cmd /c start " & fullURL, vbNormalFocus End Sub
